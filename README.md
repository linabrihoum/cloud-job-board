# Cloud Job Board

A free, fresh, well-designed job board for cloud, SRE, and DevOps roles.
Listings are curated and pulled from legitimate feeds, always show their
posting date, and link straight to the company's original posting — no
paywall for job seekers, ever. Static site built with Next.js; no database
in the MVP.

**Everything about this project lives in [docs/](./docs/):** start with
[PURPOSE.md](./docs/PURPOSE.md) for the why and
[ROADMAP.md](./docs/ROADMAP.md) for what's next.

## Run it locally

```bash
npm install
npm run dev     # http://localhost:3000
npm run lint    # ESLint
```
